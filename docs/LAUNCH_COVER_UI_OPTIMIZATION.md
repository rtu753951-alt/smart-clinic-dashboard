### ✅ **4. 細節清理與美化**

#### 按鈕優化

**優化項目**：

- ✅ **微弱發光效果**

  - `box-shadow`: 多層發光陰影
  - 內發光：`inset 0 1px 0 rgba(255, 255, 255, 0.2)`
  - 外發光：`0 0 40px rgba(59, 130, 246, 0.2)`

- ✅ **Hover 互動增強**

  - 上浮 3px
  - 發光強度提升
  - 箭頭右移動畫

- ✅ **內部發光層**
  - 使用 `::before` 偽元素
  - Radial gradient 微光效果
  - Hover 時淡入

**CSS 實作**：

```css
.launch-enter-btn {
  box-shadow: 0 4px 20px rgba(59, 130, 246, 0.4), 0 0 40px rgba(59, 130, 246, 0.2),
    inset 0 1px 0 rgba(255, 255, 255, 0.2);
}

.launch-enter-btn:hover {
  transform: translateY(-3px);
  box-shadow: 0 8px 30px rgba(59, 130, 246, 0.5), 0 0 60px rgba(59, 130, 246, 0.3);
}
```
