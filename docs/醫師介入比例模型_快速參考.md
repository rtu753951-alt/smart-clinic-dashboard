# 醫師介入比例模型 - 快速參考

## 🎯 核心概念

**醫師 = 斷點式介入資源，非全程陪同**

## 📊 介入比例表

| 療程類型           | 醫師 | 諮詢師 | 美療師 | 護理師 |
| ------------------ | ---- | ------ | ------ | ------ |
| **Inject** (注射)  | 35%  | -      | -      | -      |
| **RF** (電波)      | 35%  | -      | -      | -      |
| **Laser** (雷射)   | 15%  | -      | 100%   | -      |
| **Drip** (點滴)    | 10%  | -      | -      | 100%   |
| **Consult** (諮詢) | 30%  | 70%    | -      | -      |

## ⏰ 每日可用工時

- **醫師**: 6 小時/天 (保守估計)
- **其他職務**: 8 小時/天

## 💡 計算範例

### Botox (20 分鐘)

- 醫師: 20 × 0.35 = **7 分鐘**
- 護理師協助: 20 × 0.25 = **5 分鐘**

### Pico Laser (40 分鐘)

- 醫師: 40 × 0.15 = **6 分鐘**
- 美療師: 40 × 1.0 = **40 分鐘**

### Consultation (25 分鐘)

- 醫師: 25 × 0.30 = **7.5 分鐘**
- 諮詢師: 25 × 0.70 = **17.5 分鐘**
