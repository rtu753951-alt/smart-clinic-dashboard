# 人力負載卡片式顯示 - 快速參考

## 🎯 核心改進

### 從圖表改為卡片

- ✅ 更易讀、更直觀
- ✅ 一眼看懂負載情況
- ✅ 大字體顯示關鍵數據

## 📊 卡片內容

每張卡片顯示：

- **職務名稱** + 圖標
- **實際工時** (小時)
- **任務數量**
- **負載率** (大字體 + 進度條)
- **狀態標籤** (輕鬆/適中/偏高/過載)
- **總工時**

## 🎨 狀態指示

| 負載率 | 狀態 | 顏色    |
| ------ | ---- | ------- |
| < 40%  | 輕鬆 | 🟢 綠色 |
| 40-69% | 適中 | 🟣 紫色 |
| 70-89% | 偏高 | 🟠 橙色 |
| ≥ 90%  | 過載 | 🔴 紅色 |

## ⏰ 週期選擇

- **本週**: 本週一~週日
- **下週**: 下週一~週日
- **未來 30 天**: 從今天起 30 天
